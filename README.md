# Pokémon Typen Alexa Skill

Mitten in einem hitzigen Pokémon-Kampf und du weißt nicht mehr welcher Typ gegen Gift effektiv ist? Kein Problem! Frag einfach Alexa und behalte die Oberhand über deine Rivalen.

- Alexa frag Professor Eich welcher Typ gut gegen Feuer ist
- Alexa frag Professor Eich welche Attacke gegen Stahl hilft
- Alexa frag Professor Eich was ist effektiv gegen Boden

*Bald im Alexa Skill Store*
